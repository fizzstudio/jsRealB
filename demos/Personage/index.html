Text generation with Big Five personality-based user adaptation

A jsRealB adaptation of the ideas put forward in Towards personality-based user adaptation: psychologically informed stylistic language generation, by F. Mairesse and M. A. Walker, User Modeling and User-Adapted Interaction, 20(3):227â€“278, 2010. [PDF]

Unfortunately, the available data, found in the SIGGEN Datasets for NLG page, only deals with a subset of the 5 traits (agree, consc, extra) and the paper is not explicit about the formulations used for each trait. Some were then determined by looking at the training data and others taken from the e2eChallenge demo. We also looked at the source code of the original program, but we could not manage to run the version that created the data which was originally designed to develop text generators using learning algorithms. In our case, we use a rule-based approach with some random choices to provide variations.
