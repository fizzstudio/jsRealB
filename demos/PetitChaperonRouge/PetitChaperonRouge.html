Reproduction d'un conte

Voici une reproduction verbatim d'un conte célèbre avec sa moralité. On constate que jsRealB permet de réaliser des textes relativement complexes. Il suffit de placer la souris sur une phrase pour afficher la structure jsRealB permettant de réaliser cette phrase soit sous forme de:

ou de

À noter que l'expression affichée sous forme de constituents n'est pas identique à celle qu'on retrouve dans le code source qui utilise parfois des fonctions auxiliaires pour factoriser des parties de code, alors qu'ici l'expression est reconstruite à partir des informations sauvées lors des appels de constructeurs et de fonctions. L'expression affichée sous forme de dépendances est une transformation automatique de l'expression originale en constituents.

Le fichier JavaScript utilisé par cette page est exécutable avec node pour obtenir la version texte du conte, mais sans l'affichage sélectif des expressions jsRealB..
