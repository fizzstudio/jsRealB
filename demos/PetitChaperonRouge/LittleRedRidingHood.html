Fairy tale realization

This is a verbatim reproduction of a well known fairy tale. This illustrates that jsRealB allows the realization of relatively complex texts. Hovering with the mouse on a sentence displays the jsRealB structure to realize that sentence in either

or

The displayed expression in constituencies is not necessarily identical to the one found in the source code that often uses auxiliary functions to factorize the code, while here the expression is rebuilt from informations saved when the function is called. The dependencies version is an automatic conversion of the original constituency expression.

The JavaScript code used by this page can be executed with node to get the text version of this story, but without the selective display of the jsRealB expressions.
