This page loads the current webpack module. The console displays, the version of jsRealB, the date the webpack module has been created and the current realization language. If Constituent.debug is true, the word debug is displayed.

It is useful for developing using the JavaScript console that is displayed using the following key strokes.

  • Chrome : ctrl+shift+j (macOS: cmd+opt+j)
  • Firefox : ctrl+shift+k (macOS: cmd+opt+k) for Firefox
  • Safari : cmd+opt+c

In the console, at the prompt, jsRealB expression can be typed and its realization displayed either by calling .toString() or concatenating with an empty string. If debug is displayed, realization must be obtained with .realize().
