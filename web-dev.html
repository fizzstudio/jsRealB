This is single page that loads tests-dev.js which itself loads the current webpack module.

These tests are checked against the expected value. When the realization is not the expected one, the expression, the output and what was expected are displayed in the JavaScript console. The number of differences is also displayed.

A warning of a bad value for an option (in French) is also realized, to exercise the warning system.

jsRealB documentation

It is useful for developing using the JavaScript console that is displayed using the following key strokes.

  • Chrome : ctrl+shift+j (macOS: cmd+opt+j)
  • Firefox : ctrl+shift+k (macOS: cmd+opt+k) for Firefox
  • Safari : cmd+opt+c
